## Selected active work

| Project | System |
|---|---|
| [`token-firewall-team`](https://github.com/WdBlink/token-firewall-team) | Delegates implementation to lower-cost workers and reserves frontier-model tokens for bounded review, with reproducible evaluation evidence. |
| [`autoresearch-paper`](https://github.com/WdBlink/autoresearch-paper) | Runs long-horizon paper projects with frozen evaluators, research gates, watchdogs, and resumable state. |
| [`daily-cockpit`](https://github.com/WdBlink/daily-cockpit) | A local-first Obsidian workspace that protects today, captures ideas without pressure, and reconnects yesterday's context. |
